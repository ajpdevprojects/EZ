# Purpose

The Design Canon is the permanent visual blueprint of Ez.

It defines how the product should look, feel, move, and communicate.

This document ensures every engineer, designer, AI model, and future
contributor builds a consistent product.

This canon is the single source of truth for all visual decisions.

------------------------------------------------------------------------

# Design Philosophy

Ez is not a traditional job board.

It is a premium AI-powered Career Operating System.

Every visual decision must reinforce:

-   Confidence
-   Professionalism
-   Simplicity
-   Trust
-   Intelligence
-   Elegance

Users should immediately feel that Ez is premium software built for
serious professionals.

The interface should never feel playful, cluttered, or overwhelming.

Instead, it should feel like sitting at an executive's desk.

------------------------------------------------------------------------

# Design Direction

Executive • Elegant • Premium • Timeless • Minimal • Calm •
Sophisticated • Human • Beautiful • Intelligent

Every screen should feel intentional. Nothing exists simply for

# Typography

Primary: **Inter**

Alternative: **SF Pro Display**

Rules:

-   Large headings
-   Comfortable reading
-   Generous spacing
-   High contrast
-   No decorative fonts

------------------------------------------------------------------------

# Spacing

Base unit: **4px**

Scale:

4 • 8 • 12 • 16 • 20 • 24 • 32 • 40 • 48 • 64

Whitespace is a design element.

------------------------------------------------------------------------

# Motion

Purposeful.

150--300ms.

Motion communicates state---not decoration.
